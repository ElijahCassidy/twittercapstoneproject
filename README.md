# twittercapstoneproject
In this project, we will create a:
1. Visual layout
1. Feed formatting
1. Buttons/menus
1. Color, spacing, readability
for our class' twitter project.

## File key
- **"!!!most_updated_version"** contains the full code for a twitter clone to help us figure out how the graphics should look.
- **"!!background_color_code_only"** contains the code for the **graphics only**.
